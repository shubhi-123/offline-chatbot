## Project Overview

This project is a privacy-focused AI chatbot that runs entirely on a local machine using the **Gemma 3B** Large Language Model through **Ollama**. It enables users to have real-time conversations without relying on cloud APIs or an internet connection, ensuring complete data privacy and offline accessibility. The chatbot features a simple Streamlit interface for interacting with the locally hosted model.


## Features

* Runs the Gemma 3B language model locally using Ollama.
* Supports completely offline AI conversations with no external API calls.
* Provides low-latency inference for real-time responses.
* Interactive Streamlit interface for user-friendly chatting.
* Ensures complete privacy by processing all requests locally.


## Tech Stack

* **Language:** Python
* **Framework:** Streamlit
* **LLM:** Gemma 3B
* **Inference Engine:** Ollama


## Installation

1. Clone the repository

```bash
git clone https://github.com/shubhi-123/offline-chatbot.git
cd offline-chatbot
```

2. Install the required dependencies

```bash
pip install -r requirements.txt
```

3. Start Ollama

```bash
ollama serve
```

4. Download the Gemma 3B model

```bash
ollama pull gemma:3b
```


## Usage

Run the Streamlit application:

```bash
streamlit run app.py
```

Then open the local URL generated by Streamlit in your browser and start chatting with the locally hosted AI assistant.
